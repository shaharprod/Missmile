## Codebase Patterns
- Remotion scenes are in `remotion-missile/src/scenes/`, components in `components/`, shared constants in `lib/constants.ts`
- Scene durations are defined in frames in `constants.ts`, total frames computed from all scenes minus transitions
- Each scene is a React component using `useCurrentFrame()` and `useVideoConfig()` for animation
- 3D scenes use `@remotion/three` ThreeCanvas with a CameraController component using `useThree()`
- 2D map scenes use SVG rendering via `MilitaryMap` component
- Audio layers are managed in `MissileVideoComposition.tsx` with `Sequence` + `Audio` components
- TransitionSeries handles scene ordering and fade transitions
- Color palette defined in `C` object in constants.ts (HUD green, red, amber theme)
- Scene start frame offsets are computed in MissileVideoComposition.tsx for audio sync
